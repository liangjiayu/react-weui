## API

| 参数           | 说明             | 类型                      | 默认值 |
| -------------- | ---------------- | ------------------------- | ------ |
| checked        | 指定当前是否选中 | `boolean`                 | false  |
| className      | Switch 器类名    | `string`                  | -      |
| defaultChecked | 初始是否选中     | `boolean`                 | false  |
| disabled       | 是否禁用         | `boolean`                 | false  |
| onChange       | 变化时回调函数   | `(checked,event) => void` | -      |
| onClick        | 点击时回调函数   | `(event) => void`         | -      |
