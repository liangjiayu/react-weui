## API

| 参数        | 说明     | 类型                                       | 默认值 |
| ----------- | -------- | ------------------------------------------ | ------ |
| icon        | 图标     | `success` \| `info` \| `warn` \| `waiting` | -      |
| title       | 标题     | `ReactNode`                                | -      |
| description | 描述     | `ReactNode`                                | -      |
| operation   | 操作区域 | `ReactNode`                                | -      |
| tips        | 提示区域 | `ReactNode`                                | -      |
| extra       | 额外区域 | `ReactNode`                                | -      |
