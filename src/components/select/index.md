## API

| 参数         | 说明                 | 类型              | 默认值 |
| ------------ | -------------------- | ----------------- | ------ |
| defaultValue | 默认值               | `string`          | -      |
| value        | 选择框值             | `string`          | -      |
| onChange     | 选择框内容变化时触发 | `() => void`      | -      |
| children     | 原生`option`         | `React.ReactNode` | -      |
