## API

| 参数         | 说明                   | 类型       | 默认值 |
| ------------ | ---------------------- | ---------- | ------ |
| defaultValue | 输入框默认内容         | `string`   | -      |
| maxLength    | 内容最大长度           | `number`   | -      |
| showCount    | 是否展示字数           | `boolean`  | false  |
| value        | 输入框内容             | `string`   | -      |
| onChange     | 输入框内容变化时的回调 | `()=>void` | -      |
