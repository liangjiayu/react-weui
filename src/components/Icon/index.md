## API

| 参数    | 说明                           | 类型                                       | 默认值    |
| ------- | ------------------------------ | ------------------------------------------ | --------- |
| type    | 类型                           | `success` \| `info` \| `warn` \| `waiting` | `success` |
| size    | 图标大小                       | `string`                                   | `24px`    |
| color   | 图标颜色，如 普通警告`#ffc300` | `string`                                   | -         |
| onClick | 点击事件                       | `()=>void`                                 | -         |
