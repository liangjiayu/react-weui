## API

| 参数     | 说明               | 类型                             | 默认值    |
| -------- | ------------------ | -------------------------------- | --------- |
| type     | 按钮类型           | `primary` \| `default` \| `warn` | `default` |
| size     | 按钮大小           | `mini`                           | -         |
| block    | 是否为块级元素     | `boolean`                        | `false`   |
| disabled | 是否禁用按钮       | `boolean`                        | `false`   |
| loading  | 是否显示为加载状态 | `boolean`                        | `false`   |
| onClick  | 点击事件           | `()=>void`                       | -         |
